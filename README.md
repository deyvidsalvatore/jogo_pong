# Jogo Pong

Este é um jogo Pong simples implementado em Java.

## Descrição

O jogo Pong é um jogo clássico de arcade em que os jogadores controlam as raquetes para rebater a bola e evitar que ela passe pela sua defesa. O objetivo é marcar pontos fazendo a bola ultrapassar a defesa do oponente.

O jogo possui uma interface gráfica simples usando Java Swing para renderizar a tela e capturar as entradas do jogador.

## Como Jogar

- O jogador 1 controla a raquete da esquerda usando as teclas W (para cima) e S (para baixo).
- O jogador 2 controla a raquete da direita usando as setas para cima e para baixo.
- O objetivo é rebater a bola com as raquetes e evitar que ela ultrapasse a defesa.
- Cada vez que a bola ultrapassa a defesa do oponente, o jogador ganha um ponto.
- O jogo continua até que um jogador atinja a pontuação máxima.
- Pressione a tecla Esc para sair do jogo a qualquer momento.

## Requisitos

- Java Development Kit (JDK) 8 ou superior instalado.

## Como Executar

1. Clone o repositório:
```
git clone https://github.com/deyvidsalvatore/jogo_pong.git
```
2. Rode o PongGame.java e seja feliz =)
```
java -jar PongGame.java
```
